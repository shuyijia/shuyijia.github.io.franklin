The Vanishing Gradient Problem

Recurrent Neural Network

A recurrent neural network (RNN) is a class of artificial neural networks that is specifically designed to process sequential data. Some examples of sequential data include speeches, gene sequences, and historical stock prices.

Given an input sequence x1:T=(x1,⋯ ,xt,⋯ ,xT)\boldsymbol{x}_{1:T}=(\boldsymbol{x}_1,\cdots,\boldsymbol{x}_t,\cdots, \boldsymbol{x}_T)x1:T​=(x1​,⋯,xt​,⋯,xT​), a RNN computes the hidden vector ht\boldsymbol{h}_tht​ for the current hidden layer:

ht=f(ht−1,xt), \boldsymbol{h}_t = f(\boldsymbol{h}_{t-1},\boldsymbol{x}_t), ht​=f(ht−1​,xt​),

where f(⋅)f(\cdot)f(⋅) is a nonlinear function, ht−1\boldsymbol{h}_{t-1}ht−1​ the hidden vector calculated at the previous timestep and xt\boldsymbol{x}_txt​ the current input vector. ht\boldsymbol{h}_tht​ is also commonly referred to as the hidden state.

In the case of simple recurrent neural networks (Elman, 1990), the hidden state ht\boldsymbol{h}_tht​ is updated as follows:

ht=f(Uht−1+Wxt+b),y^t=g(Vht),\begin{aligned} \boldsymbol{h}_t &= f(U\boldsymbol{h}_{t-1} + W\boldsymbol{x}_t + \boldsymbol{b}),\\ \hat{y}_t &= g(V\boldsymbol{h}_t), \end{aligned}ht​y^​t​​=f(Uht−1​+Wxt​+b),=g(Vht​),​

where UUU and WWW are the recurrent and input weights respectively, b\boldsymbol{b}b the bias vector and y^t\hat{y}_ty^​t​ the output at timestep ttt by applying some function ggg to the hidden state.

The network weights of RNN can be learnt through the usual gradient descent method.

Exploding and Vanishing Gradients

For very long sequence length TTT, repeated multiplication of UUU in Eq. (5)(5)(5) can cause exponential growth or decay of backpropagated gradients. In particular, exploding gradients are caused by eigenvalues of UUU that are greater than 111, while vanishing gradients are caused by eigenvalues of UUU that are less than 111.

Another way of seeing this is to set γ≅∣∣diag[f′(zτ)]U∣∣\gamma \cong ||\text{diag}\left[f'(\boldsymbol{z}_\tau)\right]U||γ≅∣∣diag[f′(zτ​)]U∣∣.

If γ>1\gamma > 1γ>1 and t−k→∞t-k \rightarrow \inftyt−k→∞, then naturally we have γt−k→∞\gamma^{t-k}\rightarrow \inftyγt−k→∞, therefore the exploding gradient problem.

Conversely, if γ<1\gamma < 1γ<1 and t−k→∞t-k \rightarrow \inftyt−k→∞, we have γt−k→0\gamma^{t-k}\rightarrow 0γt−k→0, thus the vanishing gradient problem.

We can limit the exploding gradient problem by using gradient clipping. However, it is harder to obviate the vanishing gradient problem.

Long Short-term Memory

Some solutions to the vanishing gradient problem include replacing sigmoid functions σ\sigmaσ with ReLU\texttt{ReLU}ReLU and the use of gating mechanisms such as LSTM and GRU.
